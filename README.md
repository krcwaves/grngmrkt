\# GRNGMRKT — Web3 Marketplace



\## Overview

GRNGMRKT is a decentralized marketplace built on the Solana blockchain where users can buy, sell, and trade digital assets securely.



\## Features

\- Wallet Integration (Phantom / Solana wallets)

\- NFT Listing \& Trading

\- Fast and low-cost transactions

\- Decentralized ownership



\## Tech Stack

\- Frontend: HTML, CSS, JavaScript

\- Backend: Node.js

\- Blockchain: Solana



\## Installation



1\. Clone the repo:

git clone https://github.com/krcwaves/grngmrkt.git



2\. Go to project folder:

cd grngmrkt



3\. Install dependencies:

npm install



4\. Run project:

npm start
